Bug Reports

  

 

It might be fixed already

If your version of the program is more than a week old, chances are that you're not the first who spotted the bug. Since we usually fix bugs immediately and upload the fixed version to the What's New section of XYplorer's homepage within a few days, you might check there first.

 

XYplorer in trouble

If you get a bug, an error message, a crash, or some other suspicious event ... please let us know! To help us locating the problem, give us any information on the problem's context that you think might be relevant, especially:

What's the problem?

What did you do when the problem happened?

Can you reproduce the problem (if you dare)?

Can you reproduce the problem with a copy of XYplorer.exe in a temporary folder?

If you got an error message, what was it saying?

If you run this through the Address Bar <xydata>|xyerrorlog-*.txt, are there any recent error log files? If yes, what is their content?

What exact version of XYplorer are you using?

What operating system are you running?

What's the language of your system?

Is your computer part of a network neighborhood?
