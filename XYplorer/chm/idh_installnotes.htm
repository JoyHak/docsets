Further Notes on Installation

  

 

Integrating XYplorer with the Shell

To integrate XYplorer with the Windows shell check the XYplorer in shell context menu (Configuration | Shell Integration) checkbox while XYplorer is running. The new item "XYplorer" will then appear in Explorer's context menu for drives and folders.

To accomplish this, the following keys are added to the Windows registry:

HKEY_CLASSES_ROOT\Folder\Shell\XYplorer

HKEY_CLASSES_ROOT\Folder\Shell\XYplorer\Command

To remove XYplorer from the shell context menu and these keys from the registry simply uncheck the XYplorer in shell context menu checkbox.
