Quick Help: Image Preview

  

 

Mouse down on previewed image

Left Mouse DownPop up image in original size
Ctrl+Mouse DownToggle zoom/original size
Right Mouse ClickPop up context menu

 

Keys for Full Screen Preview

Shift+F11Open current image file in Full Screen Preview
Shift+F11, EscClose Full Screen Preview
PageUp/Shift+Space/WheelUpPrevious image
PageDown/Space/WheelDownNext image
Up, Down, Left, RightPan cropped image
Home/EndFirst/Last image
AFit All
WFit Width
HFit Height
O, 1Original Size
D, 2Double Size
ZToggle Zoom to Fit (Floating Preview only)
BCycle Background Color
SShow Status Bar
Ctrl+TCycle Transparency Background
QHigh Quality Image Resampling
F5Refresh
