Info Panel: Version Tab

    

 

Version Tab

Displays any version information contained in the selected file.

Pretty self-explaining: you know this stuff from the Explorer Properties dialog.

TipYou can copy virtually any information on this tab by double-clicking it. Double-clicking the key copies key and value, double-clicking the value copies just the value.
NoteIf the version tab is empty, the selected file has no version info. Version information is typically contained in EXE, DLL, DRV, OCX etc. files.
